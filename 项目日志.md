# 每日学习推送程序 - 项目日志

## 项目概述

构建一个每日学习推送程序，通过 AI 生成学习内容，定时推送到微信。

**技术栈：** Python + DeepSeek API + PushPlus 微信推送

---

## 2026-07-20 开发记录

### 完成的功能

1. **基础推送打通**
   - PushPlus 微信推送集成
   - DeepSeek AI 内容生成
   - 配置文件管理（config.yaml）

2. **内容排版优化**
   - HTML 卡片式排版（替代 Markdown）
   - 关键词标签高亮显示
   - 正文重点术语加粗
   - 知识点之间虚线分隔，增加留白
   - 配图改为 LoremFlickr 关键词匹配（真实相关图片）

3. **连载式阅读**
   - AI 智能分片（根据内容复杂度自动决定篇数）
   - 每次推送 1 篇，标注进度条（如 ██░░ 第2/5篇 40%）
   - 本地进度追踪（progress.json）
   - 每篇开头带「前情提要」衔接上下文
   - 章节完结自动发送总结回顾

4. **自动化推送**
   - 每天 8:00 和 19:00 各推一篇
   - 章节推完后自动进入下一章（无需手动干预）
   - 支持 `--daemon` 后台定时模式

### 文件结构

```
01-项目Projects/每日学习推送/
├── config.yaml           # 配置文件（token、API key、推送时间等）
├── main.py               # 主程序入口
├── pusher.py             # PushPlus 推送 + HTML 渲染
├── content_generator.py  # AI 内容生成（支持单篇/连载）
├── progress.py           # 阅读进度追踪
├── progress.json         # 本地进度数据（自动生成）
├── requirements.txt      # Python 依赖
└── preview.html          # 排版预览文件
```

### 使用方法

```bash
# 开始新书连载（只需执行一次）
python main.py --book "中医内科学评讲" --chapter "一"

# 手动推下一篇
python main.py

# 后台定时推送（每天 8:00 / 19:00）
python main.py --daemon

# 重置进度
python main.py --reset
```
